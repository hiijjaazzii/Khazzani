# Khazzani
IoT-Based Smart Water Tank Monitoring System
Designed and developed an Internet of Things (IoT) system using Arduino integrated with the Blynk IoT platform for real-time water tank monitoring.
Implemented sensors to measure water level, water quality (pollution/turbidity), and water temperature.
Enabled real-time data transmission from the Arduino device to the Blynk mobile application over the Internet.
Developed a user-friendly mobile dashboard to display live sensor readings and monitor tank conditions remotely.
Integrated multiple sensors and microcontroller components to ensure accurate environmental monitoring.
Applied embedded systems, IoT communication, sensor integration, and real-time data visualization concepts throughout the project.
Technologies Used: Arduino, Blynk IoT, C/C++, IoT Sensors, Embedded Systems, Mobile Dashboard.
